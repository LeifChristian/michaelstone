# Lyrics source files

Drop Word `.docx` files here. They are parsed at build time with [mammoth](https://github.com/mwilliamson/mammoth.js) and appear on the **Lyrics** page.

Current songs with lyrics in the zip:

- The Deal Michael.docx
- Soldier Michael.docx
- Road to Redemption Michael.docx
- Renaissance Man Michael.docx

Add more docx files for the remaining tracks when you have them. Plain `.txt` can also be added manually to `content/lyrics/` if you prefer to edit by hand (future enhancement).

After adding or changing files, run `npm run dev` or `npm run build` to regenerate.
