# Photographs

Drop image files here. They are picked up automatically (no config edit).

## Naming / order

Use a number prefix so order is predictable:

```
hero.jpg          ← homepage hero (not shown in gallery)
01-beach.jpg
02-studio.jpg
03-friends.jpg
10-later.jpg
```

Natural sort: `1`, `2`, `10` order correctly.

Supported: `.jpg` `.jpeg` `.png` `.webp` `.gif` `.avif`

Hero reserved names: `hero.*`, `cover.*`, `portrait.*`
